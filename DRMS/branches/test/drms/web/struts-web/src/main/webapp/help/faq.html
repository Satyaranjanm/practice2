Account Manager
Q: Can I opt out or force DR control for my customer facility?
A: No. You don't have access to customer manual control.
Q: I mistyped customer's Service Agreement ID. How can I correct it?
A: No, you cannot change Service Agreement ID after the first input. Delete the mistyped one, and create new SAID. Once you create a new SAID for a replacement, contact the installation coordinator to reconfigure the control information.
Facility Manager
Q: I don't see only one building in my login, even though I participate two buildings.
A: If you have separate communication devices (software client, CLIR box, or Internet relay) for each building, you have separate username and password for each building. To see the status of both buildings, you have to login separately for each. Ask your installation coordinator if you don't remember username and password of the other building.
Q. My CLIR box indicates "Communication Good", but DRAS says my CLIR hasn't been communicating. Or vice versa. Which is true?
A: Reboot CLIR box, and see how it goes. If the mismatch continues, contact your installation coordinator.
Q: My communication has stopped after I changed my login password.
A: The login username and password for DRAS are same as your software client / CLIR box username and password. Once you changed your password on DRAS, you need to change password of software client / CLIR as well.
Installation Coordinator
Q: How to change Comm Dev name?
A: No, you cannot change Comm Dev name after the first input. Delete the mistyped one, and create new Comm Dev name. Reconfigure all the relative input.
